MailYou 服务条款 ​

生效日期：2026-03-16

欢迎使用 MailYou。使用 MailYou 即表示你同意遵守本服务条款。

1. 服务说明 ​

MailYou 是一款桌面邮件客户端，提供邮箱接入、邮件同步、邮件发送、联系人管理及 OAuth 登录等相关能力。

MailYou 可能通过以下方式接入第三方服务：

  • 标准 IMAP / SMTP 邮件服务
  • Gmail、Outlook、iCloud 等 OAuth 身份认证服务
  • MailYou OAuth 代理

2. 用户责任 ​

你同意：

  • 仅在你有权访问和使用相关邮箱账户时使用 MailYou。
  • 遵守适用法律法规，以及相关邮箱服务商、OAuth 提供商的服务条款和政策。
  • 不利用 MailYou 从事垃圾邮件、欺诈、未授权访问、恶意扫描、批量滥用或其他违法违规行为。
  • 妥善保管你的设备、邮箱凭据、应用专用密码、OAuth 授权信息及本地数据。

3. 第三方服务 ​

MailYou 依赖第三方邮箱服务和身份认证服务。此类服务的可用性、稳定性、接口行为、限流策略和账户政策由相应第三方决定。

对于因第三方服务故障、限流、封禁、接口变更、授权撤销或账户异常导致的功能受限、同步失败或数据不可用，MailYou 不作绝对保证。

4. OAuth 与 Google API 使用 ​

当你使用 Google 账号或 Gmail 相关能力时，你还应遵守 Google 的相关条款和政策，包括 Google API Services User Data Policy。

你理解并同意：

  • MailYou 仅在你主动授权后访问相应账户数据。
  • 你可以随时在提供方账户设置中撤销授权。
  • 你撤销授权后，相关 OAuth 功能可能无法继续使用。

5. 数据与隐私 ​

MailYou 如何处理你的数据，见：

  • MailYou 隐私政策

6. 免责声明 ​

MailYou 按“现状”提供。在适用法律允许的最大范围内，我们不对以下事项作明示或默示保证：

  • 服务始终不中断、无错误或完全符合你的特定需求；
  • 任意第三方邮箱服务始终兼容；
  • 任意邮件、附件、联系人或本地数据在所有情况下都不会丢失、损坏或延迟。

你应自行负责对重要数据进行备份，并自行评估在生产或个人重要通信场景中使用本软件的风险。

7. 责任限制 ​

在适用法律允许的最大范围内，对于因使用或无法使用 MailYou 而产生的任何间接损失、附带损失、特殊损失、后果性损失、数据损失、业务中断或利润损失，我们不承担责任。

我们可能根据产品功能变化、法律要求或第三方平台政策更新本服务条款。更新后的版本将通过公开页面发布，并以新的生效日期为准。
