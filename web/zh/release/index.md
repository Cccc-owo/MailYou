# 下载 MailYou

MailYou 的发布包统一放在 GitHub Releases。

## 获取最新版本

- [打开最新版本](https://github.com/Cccc-owo/MailYou/releases/latest)
- [查看全部发布](https://github.com/Cccc-owo/MailYou/releases)
- [查看 GitHub Actions 测试构建](https://github.com/Cccc-owo/MailYou/actions/workflows/release.yml)

## 当前提供的包

MailYou 目前提供这些桌面构建产物：

- Linux AppImage
- Linux deb
- Linux rpm
- Linux zip
- Windows exe
- Windows zip

## 说明

- 请按你的系统选择对应安装包。
- 对大多数 Linux 用户来说，AppImage 是最直接的选择。
- 版本说明和附件变更以 GitHub 页面为准。
- 如无特殊需要，建议优先下载 GitHub Releases 中的正式发布版本。
- GitHub Actions 中提供的是测试构建，用于验证和抢先体验；它们可能随时变化，也可能包含尚未完成的改动。
