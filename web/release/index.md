# Download MailYou

MailYou releases are published on GitHub Releases.

## Get the latest build

- [Open the latest release](https://github.com/Cccc-owo/MailYou/releases/latest)
- [Browse all releases](https://github.com/Cccc-owo/MailYou/releases)
- [Open GitHub Actions test builds](https://github.com/Cccc-owo/MailYou/actions/workflows/release.yml)

## Available packages

MailYou currently publishes desktop builds for:

- Linux AppImage
- Linux deb
- Linux rpm
- Linux zip
- Windows exe
- Windows zip

## Notes

- Pick the package that matches your system.
- AppImage is the simplest option for most Linux users.
- Release notes and asset changes are maintained on GitHub.
- GitHub Releases are the recommended stable download source.
- GitHub Actions artifacts are test builds for validation and early access. They may change at any time, and may contain unfinished work.
