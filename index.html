MailYouA refined local-first mail client for desktop

MailYou combines a comfortable desktop experience with local storage, provider presets, and modern OAuth support.

Quick Overview
中文介绍

Desktop-first

Built for desktop use with Vue, Electron, and a Rust backend rather than a browser-based mail workflow.

Fast account setup

Common providers are preconfigured, while advanced server options remain available when manual setup is required.

OAuth ready

Supports direct OAuth credentials as well as the built-in MailYou OAuth proxy for supported providers.

Local-first storage

Mail data, drafts, and account state are designed to remain primarily on the local device.

What MailYou is ​

MailYou is a local-first desktop mail client focused on stable account access, efficient setup, and a more comfortable daily workflow.

The application is built with Vue and Electron on the desktop side, with Rust handling mail access and local persistence.

Current scope ​

  • Desktop app, not a webmail service
  • Local account storage on your machine
  • Presets for Gmail, Outlook, iCloud, QQ Mail, 163, 126, Yeah, and Foxmail
  • OAuth flow through direct credentials or the MailYou public proxy

MailYou focuses on local-first desktop email.
